# Hive Sovereign Site: Image Art-Direction Brief

Company: S Cubed Technology Ltd. Site: hive.scubed.co. Audience: general counsel, CISOs, compliance officers, and non-technical SMB owners in regulated sectors. Restrained, institutional, credible. Not generic AI stock.

## Global style rules (baked into every prompt below)

These constraints are repeated inside each generation prompt so each image stands alone:

- Abstract and restrained. Quiet institutional trust, Canadian restraint, understated.
- Palette only: ink #0d1117, deep forest green #0d3b2e, warm off-white #f8f7f4. A single faint teal #2dd4bf accent is permitted as a containment or boundary cue, used sparingly. Gold is reserved for one hero accent only, used at most once across the whole set.
- No literal people, no faces, no hands.
- No padlocks, no shields, no keyholes, no hooded figures, no hackers, no glowing brains, no neural-net filament balls, no binary rain, no circuit-board cliches, no robot imagery.
- No text, no numbers, no logos, no UI chrome inside the image.
- Evoke data held within a boundary, contained networks, an in-region datacentre rendered as calm abstract geometry, a closed perimeter with nothing leaving it.
- Composition: generous negative space, calm symmetry or deliberate asymmetry, soft directional light, matte finish, fine grain. Editorial, architectural, museum-quiet. Must not look like a tech-startup stock render.
- Hex-lattice or hexagonal-cell motif is the recurring on-brand geometric language (the site uses a faint hex lattice). Use it subtly, never as a busy pattern.

## Cross-cutting placement note

The gold accent is permitted in SLOT 1 only, used once, consistent with the site's one-gold-per-viewport rule. Every other slot stays ink, forest green, warm off-white, with the faint teal boundary cue. Keeping the whole set on the same restrained geometric language is what separates it from generic AI stock and reads as a single institutional brand.
