# RBAC-18-02 — table-driven matrix test (the spec readable as "what each role can do")

## Scope

`policy_test.go` contains `TestPolicyMatrix` — a 55-case table-driven test that
serves as the human-readable authoritative specification of the permission matrix.
Each row names the actor state (role + verified flag) and the permission, and
asserts the `Policy.Can` decision. This test is the contract: if it changes, the
decision function changed. It is the primary regression guard for the RBAC model.
