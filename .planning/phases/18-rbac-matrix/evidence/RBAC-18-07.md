# RBAC-18-07 — lint-no-bare-role-check.mjs CI blocking step

## Scope

`packages/openai-contract/scripts/lint-no-bare-role-check.mjs` is a Node.js script
(mirroring the shape of `lint-no-customer-usd.mjs` from Phase 17) that scans the
repository for bare role and email-verification checks outside the authz package
allowlist. `FORBIDDEN_PATTERNS` covers `chosen.Role == "owner"`, `.EmailVerified &&`,
`IsPlatformAdmin\b`, and equivalent TS patterns. `ALLOWLIST_DIRS` covers
`internal/authz/`, `internal/platform/`, `actor_resolver.go`, `service.go` (adapter
sites), and matrix test files. The script exits non-zero on any violation. It is
wired into the `web-unit` CI job as a blocking required step.
