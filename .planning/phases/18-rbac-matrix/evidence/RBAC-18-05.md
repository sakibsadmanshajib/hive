# RBAC-18-05 — TS codegen: permissions.generated.ts from Go registry, CI drift check

## Scope

A Go emitter program `apps/control-plane/cmd/gen-permissions/main.go` consumes
`authz.AllPermissions()` and writes `apps/web-console/lib/control-plane/permissions.generated.ts`
containing an `as const` tuple `PERMISSIONS` and a `Permission` union type. A `Makefile`
target `gen-permissions` wraps the emitter. CI (`.github/workflows/ci.yml`) runs
`make gen-permissions && git diff --exit-code` as a blocking step so any registry
change that is not re-generated fails the build.
