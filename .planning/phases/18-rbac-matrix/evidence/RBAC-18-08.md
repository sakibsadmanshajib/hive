# RBAC-18-08 — Go integration matrix coverage (role × verified × module) on full control-plane suite

## Scope

Each of the 8 migrated handler packages gained table-driven `TestHandler_*AuthzMatrix`
tests covering the (role × verified) cross-product for their specific permission. The
tests exercise the HTTP handler boundary (not just the policy function), asserting
correct HTTP status codes (200/201 vs 403) for each actor state. Together with
`TestPolicyMatrix` in the authz package, these provide integration-level matrix
coverage across the full control-plane suite (23 packages, all passing).
