# RBAC-18-01 — authz package: Permission enum + registry + Policy.Can decision function

## Scope

A new Go package `apps/control-plane/internal/authz` defines the complete
permission surface for Hive v1.1: 11 typed `Permission` string constants, a
`permissionRegistry` with per-permission `RequiresVerified` flags, `AllPermissions()`
for registry enumeration, an `Actor` struct (role + verified), a stateless
`Policy.Can(actor, perm) bool` decision function, and a `RequirePermission(perm)`
HTTP middleware. This is the single authoritative source for all authz decisions
in the control-plane.
