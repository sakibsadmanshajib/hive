    1. `accountID, err := h.accountsSvc.EnsureViewerContext(r.Context())`
    2. Parse JSON body: `{"rail": "stripe", "credits": 100000, "idempotency_key": "user-checkout-abc123"}`
       ```go

           Rail           Rail   `json:"rail"`
       ```

    6. Map errors:
       - `ErrBillingProfileRequired` -> 400 `{"error": {"message": "Complete billing profile required before first purchase", "type": "invalid_request_error"}}`
       - `ValidationError` -> 400
       - Other -> 500

       ```json
       {
           "payment_intent_id": "uuid",
           "redirect_url": "https://...",
           "rail": "stripe",
           "credits": 100000,
           "amount_usd": 100,
           "amount_local": 0,
           "local_currency": "",
           "tax_treatment": "no_tax",
           "expires_at": "2026-04-10T..."
       }
       ```

       ```go
       headers := make(map[string]string)
       ```
