    **fx.go** - FXService:

    - `type FXService struct` with fields: `httpClient *http.Client`, `xeAccountID string`, `xeAPIKey string`, `redisClient *redis.Client`, `adminOverrideRate string` (empty = no override)
    - `func NewFXService(httpClient *http.Client, xeAccountID, xeAPIKey string, redisClient *redis.Client) *FXService`
    - `func (f *FXService) SetAdminOverride(rate string)` — sets or clears the admin override
    - `func (f *FXService) FetchUSDToBDT(ctx context.Context) (midRate string, sourceAPI string, err error)`:
      1. If adminOverrideRate is non-empty, return (adminOverrideRate, "admin_override", nil)
      2. Call XE API: GET `https://xecdapi.xe.com/v1/convert_from.json/?from=USD&to=BDT&amount=1` with HTTP Basic Auth (xeAccountID, xeAPIKey), 10s timeout
      3. Parse JSON response: extract `to[0].mid` as string
      4. On success: cache in Redis key `fx:usd_bdt:mid_rate` with 5-minute TTL, return (rate, "xe", nil)
      5. On XE failure: read from Redis key `fx:usd_bdt:mid_rate`. If found, return (cachedRate, "cache", nil). If not found, return ("", "", ErrFXUnavailable)
    - `func (f *FXService) CreateSnapshot(ctx context.Context, repo Repository, accountID uuid.UUID) (FXSnapshot, error)`:
      1. Call FetchUSDToBDT
      2. Compute effectiveRate = midRate * (1 + 0.05) — use string-based decimal math or `math/big` to avoid float corruption. Parse midRate to `*big.Rat`, multiply by `big.NewRat(105, 100)`, format back to string with 6 decimal places.
      3. Insert FXSnapshot row via repo.InsertFXSnapshot
      4. Return the snapshot
