# PAY-14-10 — Phase 18 RBAC contract stub via platform.IsWorkspaceOwner + IsPlatformAdmin

## Truth

Owner-gating is centralised behind a single primitive so Phase 18's
tier-aware permission matrix can swap the body without breaking any
call-site. Every mutating handler in Phase 14 invokes the primitive
exactly once at the top.

## Implementation

- `apps/control-plane/internal/platform/role.go` exports the contract:
  - `RoleService.IsWorkspaceOwner(ctx, userID, workspaceID) (bool, error)`
  - `RoleService.IsPlatformAdmin(ctx, userID) (bool, error)`
  - `RequirePlatformAdmin` middleware — wraps owner-gated routes.
- Concrete pgx-backed implementation in `role_pgx.go` reads the
  `accounts.users.is_platform_admin` flag installed by migration
  `20260428_01_budgets_alerts_invoices_grants.sql`.
- Phase 18 hand-off: replace the flag check with the tier-aware matrix;
  the public function signatures remain stable.
