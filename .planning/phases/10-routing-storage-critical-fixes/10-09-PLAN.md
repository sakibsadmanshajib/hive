<interfaces>
Control-plane accounting accepts exactly these policy modes:

```go
const (
	PolicyModeStrict           PolicyMode = "strict"
	PolicyModeTemporaryOverage PolicyMode = "temporary_overage"
)
```

`validateCreateReservation` rejects blank `model_alias` and rejects any `policy_mode` other than `strict` or `temporary_overage`.

The batch handler must extend its local reservation contract to:

```go
type ReservationInput struct {
	AccountID        string
	APIKeyID         string
	RequestID        string
	Endpoint         string
	ModelAlias       string
	EstimatedCredits int64
}
```
</interfaces>
