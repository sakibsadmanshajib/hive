  <action>
    In `apps/edge-api/cmd/server/main.go`, import `github.com/hivegpt/hive/packages/storage`.

    Add:
```go
type storageConfig struct {
	Endpoint     string
	AccessKey    string
	SecretKey    string
	Region       string
	FilesBucket  string
	ImagesBucket string
}
```

    Add `loadStorageConfigFromEnv() (storageConfig, error)` that trims and validates:
    - `S3_ENDPOINT`
    - `S3_ACCESS_KEY`
    - `S3_SECRET_KEY`
    - `S3_REGION`
    - `S3_BUCKET_FILES`
    - `S3_BUCKET_IMAGES`

    Error strings must contain the exact missing variable name plus `is required`, for example `S3_REGION is required`.

    Replace the current optional storage initialization block with:
```go
storageCfg, err := loadStorageConfigFromEnv()
if err != nil {
	log.Fatalf("storage unavailable: %v", err)
}
storageClient, err := storage.NewS3Client(storage.Config{
	Endpoint:  storageCfg.Endpoint,
	AccessKey: storageCfg.AccessKey,
	SecretKey: storageCfg.SecretKey,
	Region:    storageCfg.Region,
})
if err != nil {
	log.Fatalf("storage unavailable: %v", err)
}
```

    Do not log warnings and skip route registration. Storage is required.
  </action>
