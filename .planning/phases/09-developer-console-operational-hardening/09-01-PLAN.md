Build all new and extended control-plane API endpoints that the console pages will consume: analytics aggregation (usage, spend, errors grouped by model/key/endpoint/time), invoice list and detail, budget threshold CRUD, cursor-based ledger pagination, and a public model catalog endpoint.

Purpose: Console pages in Plans 02 and 03 need backend data sources. Without these endpoints, the console would have to aggregate raw events in JavaScript (an anti-pattern) and would lack invoice/budget data entirely.

Output: 7 new API endpoints registered in the control-plane router, 2 new DB migrations, 1 new Go package (budgets), and extensions to the existing ledger, usage, and catalog packages.

- `GetUsageSummary(ctx, filter AnalyticsFilter) ([]UsageSummaryRow, error)`:
  ```sql
  SELECT {group_column} AS group_key,
         SUM(input_tokens) AS total_input_tokens,
         SUM(output_tokens) AS total_output_tokens,
         SUM(CASE WHEN hive_credit_delta < 0 THEN ABS(hive_credit_delta) ELSE 0 END) AS total_credits_spent,
         COUNT(*) AS request_count
  FROM usage_events
  WHERE account_id = $1 AND created_at >= $2 AND created_at < $3
  GROUP BY {group_column}
  ORDER BY total_credits_spent DESC
  ```
  Where `{group_column}` is `model_alias` when GroupBy="model", `api_key_id::text` when GroupBy="api_key", `endpoint` when GroupBy="endpoint". Use a switch statement to pick the column name — do NOT interpolate user input directly into SQL. Build 3 separate query strings at the switch level and select.

- `GetSpendSummary(ctx, filter AnalyticsFilter) ([]SpendSummaryRow, error)`:
  Similar GROUP BY on `credit_ledger_entries` table: `SELECT {group_column} AS group_key, SUM(ABS(credits_delta)) AS total_credits, COUNT(*) AS entry_count FROM credit_ledger_entries WHERE account_id = $1 AND created_at >= $2 AND created_at < $3 AND credits_delta < 0 GROUP BY {group_column}`. Note: ledger entries don't have model_alias directly. For model grouping on spend, query usage_events instead and sum `ABS(hive_credit_delta)`.

- `GetErrorSummary(ctx, filter AnalyticsFilter) ([]ErrorSummaryRow, error)`:
  ```sql
  SELECT {group_column} AS group_key,
         COUNT(*) FILTER (WHERE error_code != '' AND error_code IS NOT NULL) AS error_count,
         COUNT(*) AS total_requests,
         ROUND(COUNT(*) FILTER (WHERE error_code != '' AND error_code IS NOT NULL)::numeric / NULLIF(COUNT(*), 0), 4) AS error_rate
  FROM usage_events
  WHERE account_id = $1 AND created_at >= $2 AND created_at < $3
  GROUP BY {group_column}
  ORDER BY error_count DESC
  ```
