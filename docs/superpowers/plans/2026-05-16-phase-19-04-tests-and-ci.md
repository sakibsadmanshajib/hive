> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the full Phase 19 test surface: Playwright user-flow E2E suite (7 specs, runs per push), Open WebUI direct E2E suite (10 specs + 2 performance specs, nightly + dev-time only), audit-evidence Go test suite (chain integrity, control coverage, sink failure resilience, cold-archive retention), the SOC2 coverage report generator, and the CI workflows that gate Phase 19 merges.

**Architecture:** Four test surfaces from the spec (Go unit/integration, Playwright per-push, audit-evidence, OWUI direct nightly) become four targets in CI. Two GitHub Actions workflows orchestrate them: `phase-19.yml` runs Surfaces 1–3 on every push and PR; `phase-19-owui-nightly.yml` runs Surface 4 nightly and on `ci:e2e-owui`-labelled PRs. All tests use real environments (no mocks for Supabase, OWUI, LiteLLM, providers, pgvector).

**Tech stack:** Playwright 1.x (`@playwright/test`), `pnpm` workspaces (already configured for `web-console`), Testcontainers-go (already used by Plans 01-03), Postgres + Supabase test project, GitHub Actions (`docker-compose-action` step approach via direct `docker compose` invocations), real OAuth test accounts (Google), real LLM provider keys with cheap models (`gpt-4o-mini`, `claude-3-haiku-20240307`).

- [ ] **Step 2: Write the generator**

```javascript
// tools/soc2-coverage-report.mjs
// Reads tools/soc2-control-map.yaml + queries audit_log for the last N
// minutes, emits docs/compliance/SOC2-LOG-COVERAGE.md. Exits non-zero
// when any control has zero matching events (CI gate).

import { readFileSync, writeFileSync } from 'node:fs';
import { parse as parseYAML } from 'yaml';
import pg from 'pg';

const SINCE_MIN = Number(process.env.SOC2_SINCE_MIN || 60);
const DB_URL    = process.env.HIVE_TEST_DB_URL || process.env.SUPABASE_DB_URL;
const MODE      = process.argv.includes('--check') ? 'check' : 'report';

if (!DB_URL) {
  console.error('SOC2: HIVE_TEST_DB_URL or SUPABASE_DB_URL required');
  process.exit(2);
}

const map = parseYAML(readFileSync('tools/soc2-control-map.yaml', 'utf8'));
const client = new pg.Client({ connectionString: DB_URL });
await client.connect();

const sinceLiteral = `now() - interval '${SINCE_MIN} minutes'`;
const rows = await client.query(
  `SELECT action, count(*)::int AS n FROM public.audit_log WHERE ts >= ${sinceLiteral} GROUP BY action`
);
await client.end();

const counts = Object.fromEntries(rows.rows.map(r => [r.action, r.n]));

const lines = [];
lines.push('# SOC 2 Type II — Audit Log Coverage Report\n');
lines.push(`Window: last ${SINCE_MIN} min.\n`);
lines.push('| Control | Description | Action(s) | Hits |');
lines.push('| --- | --- | --- | --- |');

let missing = 0;
for (const [ctrl, def] of Object.entries(map.controls)) {
  const total = def.actions.reduce((s, a) => s + (counts[a] || 0), 0);
  if (total === 0) missing++;
  lines.push(`| **${ctrl}** | ${def.description} | ${def.actions.join(', ')} | ${total} |`);
}

const body = lines.join('\n') + '\n';
writeFileSync('docs/compliance/SOC2-LOG-COVERAGE.md', body);

if (MODE === 'check' && missing > 0) {
  console.error(`SOC2: ${missing} control(s) have zero matching events in the window`);
  process.exit(1);
}

console.log(`SOC2: report written; ${missing} control(s) missing in last ${SINCE_MIN} min.`);
```

```go
func TestAuditChainIntegrity_AcrossPartition(t *testing.T) {
	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	defer cancel()
	pool := newPool(t, ctx)
	t.Cleanup(func() { pool.Close() })

	w := audit.NewSyncWriter(pool, audit.WriterConfig{DeploySHA: "ci", Env: "ci"})
	for i := 0; i < 120; i++ {
		require.NoError(t, w.Write(ctx, audit.Event{

			Actor: audit.Actor{Type: audit.ActorUser},

		t.Skip("HIVE_TEST_DB_URL not set")

	pool, err := pgxpool.New(ctx, dsn)

	return pool
```

```go
// tests/compliance/audit_sink_failure_test.go
package compliance_test

import (
	"context"
	"errors"
	"os"
	"testing"
	"time"

	"github.com/jackc/pgx/v5/pgxpool"
	"github.com/stretchr/testify/require"

	"hive/control-plane/internal/audit"
	"hive/control-plane/internal/auditworker"
)

func TestAuditSink_FailureDoesNotBlockChat(t *testing.T) {
	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	defer cancel()
	pool := newPool(t, ctx)
	t.Cleanup(func() { pool.Close() })

	w := audit.NewSyncWriter(pool, audit.WriterConfig{DeploySHA: "ci", Env: "ci"})
	require.NoError(t, w.Write(ctx, audit.Event{
		Action: "CHAT_REQUEST", Severity: audit.SeverityInfo,
		Actor: audit.Actor{Type: audit.ActorUser},
	}))

	_, _ = pool.Exec(ctx, `INSERT INTO public.audit_outbox(audit_id, audit_ts, sink)
	   SELECT id, ts, 'fail-sink' FROM public.audit_log WHERE action='CHAT_REQUEST' ORDER BY ts DESC LIMIT 1`)

	failSink := &funcSink{name: "fail-sink", fn: func(ctx context.Context, row map[string]any) error {
		return errors.New("simulated outage")
	}}
	worker := auditworker.New(auditworker.Config{
		Pool: pool, Sinks: []auditworker.Sink{failSink},
		MaxAttempts: 2, BackoffStart: 25 * time.Millisecond,
	})
	go worker.Run(ctx)

	require.Eventually(t, func() bool {
		var n int
		_ = pool.QueryRow(ctx, `SELECT count(*) FROM public.audit_outbox_dlq WHERE sink='fail-sink'`).Scan(&n)
		return n > 0
	}, 8*time.Second, 200*time.Millisecond, "DLQ row must appear after retry budget")
}

type funcSink struct {
	name string
	fn   func(ctx context.Context, row map[string]any) error
}

func (f *funcSink) Name() string                                       { return f.name }
func (f *funcSink) Send(ctx context.Context, r map[string]any) error   { return f.fn(ctx, r) }
```

## 5. Common pitfalls

* `AUTH_JWT_INVALID` audit rows on every chat — the OWUI pipeline filter
  did not pick up the user JWT. Confirm `PIPELINES_URLS` points at the
  mounted file. Adopt the Go sidecar fallback (see spec §6) if the OWUI
  build does not honour `__metadata.upstream_auth`.
* `403 NO_TENANT` after first sign-in — the Supabase Database Webhook
  did not fire. Inspect Supabase dashboard → Database → Webhooks; ensure
  the `auth.users INSERT` webhook is registered against
  `${CONTROL_PLANE_URL}/internal/auth/user-created`.
* Open WebUI native admin still visible at `/admin` — confirm you are
  hitting port `3003` (Caddy) not `3002` (OWUI direct). The Caddy block
  is the user-facing surface.
